# Crypt Points Tracker

A small static webpage to track Total Battle crypt points toward a weekly goal.

## Features

- Plus/minus controls per item
- Auto-calculates total points and points missing
- Goal is editable (default 15,000)
- Stores quantities locally (localStorage)
- Export to CSV

## Run locally

- Open `index.html` in a browser

## Publish on GitHub Pages

- Push these files to a GitHub repo
- In GitHub: Settings > Pages
- Source: Deploy from a branch
- Branch: `main` (or `master`) and folder `/root`
- Save, then open the provided Pages URL

## Data source

Points come from your screenshot table.
